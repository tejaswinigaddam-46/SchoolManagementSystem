# Landing Page Setup Guide

## Step 1: Install Dependencies

Open PowerShell 7 or Command Prompt in the frontend folder and run:

```bash
cd f:\Sis\SchoolManagementSystem\frontend
npm install react-helmet-async vite-plugin-ssg
```

## Step 2: Create Directory Structure

Create the following directory:

- `src/components/landing/`

You can do this manually or run:

```bash
mkdir src\components\landing
```

## Step 3: Create Component Files

Create the following files in `src/components/landing/`:

1. Hero.jsx
2. ProblemSolution.jsx
3. RoleFeatures.jsx
4. MobileAppSection.jsx
5. AIFeatures.jsx
6. HowItWorks.jsx
7. Benefits.jsx
8. DemoForm.jsx
9. Footer.jsx
10. LandingPage.jsx

(Complete code for each is provided in the next sections)

## Step 4: Create SEO Files

In the `public/` folder, create:

- sitemap.xml
- robots.txt

## Step 5: Update Configuration Files

Update the following files:

- vite.config.js
- src/main.jsx
- src/App.jsx
