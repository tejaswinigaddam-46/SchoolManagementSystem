# 🚀 Quick Start Guide

## Run the Landing Page

```bash
cd f:\Sis\SchoolManagementSystem\frontend
npm run dev
```

Then open your browser to: **http://localhost:3001/**

---

## What You'll See

✨ **Beautiful SaaS Landing Page** featuring:

- Hero section with CTA
- Problem/Solution comparison
- Role-based features (Admin, Teachers, Parents, Students)
- Mobile app showcase
- AI features (Tutor, Quiz Generator)
- How it works (3 steps)
- Benefits grid
- Demo request form
- SEO-optimized footer
